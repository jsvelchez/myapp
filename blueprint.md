# Project Blueprint: Splitzy

## Overview

Splitzy is a Flutter application designed for expense splitting. It utilizes Firebase for user authentication, allowing users to sign in, sign up, and recover their passwords. The app features a modern and clean user interface with a custom theme, incorporating the Urbanist font from Google Fonts.

## Style, Design, and Features

### Initial Version

*   **Project Setup:**
    *   Flutter project with Firebase integration (`firebase_core`, `firebase_auth`).
    *   Uses `google_fonts` for custom typography (Urbanist).
*   **Authentication:**
    *   **Login Screen:** (`lib/screens/auth/login_screen.dart`) - Allows existing users to sign in.
    *   **Signup Screen:** (`lib/screens/auth/signup_screen.dart`) - Allows new users to create an account.
    *   **Forgot Password Screen:** (`lib/screens/auth/forgot_password_screen.dart`) - Provides a way for users to reset their password.
*   **Theming:**
    *   A custom theme is defined in `lib/theme.dart`.
    *   **Primary Color:** A teal shade (`#14B8A6`).
    *   **Typography:** Uses the "Urbanist" font from `google_fonts` with different styles for headlines, subtitles, and body text.
    *   **Component Styles:** Custom styles for `ElevatedButton`, `TextButton`, and `InputDecoration` to ensure a consistent look and feel.
